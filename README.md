# 🎓 Student Performance Prediction using Machine Learning

A machine learning project that predicts student exam performance based on academic and lifestyle factors using regression techniques.

## 📌 Project Overview

Student performance depends on multiple factors such as study hours, attendance, previous marks, sleep habits, and motivation.  

This project analyzes those factors and predicts the final exam score using Machine Learning.

## 📂 Project Files

- `project-1.ipynb` → Jupyter Notebook containing complete code  
- `dataset.csv` → Student performance dataset  
- `README.md` → Project documentation

## 📊 Dataset Features

The dataset may include:

- Study Hours
- Attendance
- Sleep Hours
- Previous Scores
- Assignments Completed
- Motivation Level
- Internet Access
- Extra Classes
- Final Score (Target)
